Funcionalidade da (h1 ate h6)

(Voltar)

As tags h1 a h6 no HTML são usadas para definir títulos e subtítulos em uma página web. Elas são elementos de cabeçalho que ajudam a organizar e estruturar o conteúdo de forma hierárquica, com diferentes níveis de importância.

Em resumo, as tags h1 a h6 são essenciais para estruturar o conteúdo de uma página, melhorando a leitura, navegação e visibilidade nos motores de busca.

Funcionalidade da tag (P)

(Voltar)

A tag P no HTML é usada para definir parágrafos de texto. Sua principal funcionalidade é agrupar e separar blocos de texto de forma lógica e organizada. Ela é uma das tags mais básicas e comuns em qualquer página da web, servindo para estruturar conteúdo textual e facilitar a leitura.

A tag P permite que o texto seja organizado em blocos, ou seja, ela delimita onde começa e onde termina um parágrafo. Isso ajuda a separar ideias, pensamentos ou tópicos diferentes em partes distintas da página.

Funcionalidade da Tag (Mark)

(Voltar)

A tag Mark no HTML é usada para destacar uma parte específica do texto, como se estivesse sendo marcada com um marcador fluorescente. O objetivo é chamar a atenção para trechos importantes ou relevantes do conteúdo. Quando aplicada, a Mark exibe o texto com um fundo amarelo (por padrão) para realçar a informação.

A principal função da Mark é realçar partes do texto que precisam de atenção especial. Pode ser usada para destacar palavras-chave, frases importantes, termos técnicos ou informações que o autor quer que o leitor perceba facilmente.

Funcionalidade Tag (Small)

(Voltar)

A tag Small no HTML é usada para exibir texto em um tamanho menor do que o texto ao redor. A principal funcionalidade dessa tag é representar textos auxiliares, como notas de rodapé, avisos legais, direitos autorais, disclaimers, ou qualquer outro tipo de informação que seja importante, mas que deva ser exibida de forma menos destacada visualmente.

A tag Small reduz o tamanho da fonte do texto dentro dela, tornando-o visualmente menos proeminente em relação ao texto principal. Essa redução é relativa ao tamanho do texto ao redor.

Funcionalidade Tag (I)

(Voltar)

A tag I no HTML é usada para exibir texto em itálico. Originalmente, ela tinha a função de estilizar o texto para inclinação (itálico) sem adicionar significado específico, mas hoje em dia, sua aplicação está mais relacionada a indicar termos que precisam ser diferenciados do texto normal por motivos semânticos, como estrangeirismos, nomes de obras, ou termos técnicos.

Funcionalidade Tag (U)

(Voltar)

A tag U no HTML é usada para sublinhar texto. Originalmente, ela era utilizada apenas para estilização visual, mas com o tempo, seu uso evoluiu para representar texto que precisa ser destacado ou diferenciado do restante de forma visual, sem indicar ênfase. O texto sublinhado pode sugerir vários significados contextuais, como termos que precisam de atenção especial, erros ortográficos em editores de texto, ou termos em um idioma diferente.

A tag U aplica um sublinhado ao texto, destacando-o visualmente. Isso pode ser útil para chamar a atenção do leitor para informações específicas ou dar destaque a certos termos que precisem de uma diferenciação visual.

Funcionalidade Tag (Strong)

(Voltar)

A tag Strong no HTML é usada para dar ênfase forte a um trecho de texto, indicando que ele é importante e deve ser destacado. Por padrão, os navegadores exibem o texto dentro da strong em negrito, mas o significado da tag vai além do visual: ela também tem uma conotação semântica, sugerindo que o conteúdo é mais importante do que o texto ao redor.

A tag Strong é usada quando o autor quer destacar a importância de uma palavra ou frase específica no conteúdo, como uma advertência, um ponto essencial ou uma informação crítica.

Funcionalidade Tag (Ol)

(Voltar)

A tag (ol) no HTML é usada para criar uma lista ordenada de itens. Diferente da tag (ul), que cria uma lista não ordenada (com marcadores como pontos), a (ol) cria uma lista em que os itens são numerados ou apresentados em uma ordem sequencial específica.

Atributos úteis da (ol): type: Define o estilo de numeração (números, letras, algarismos romanos, etc.). type="1": Números (padrão) type="A": Letras maiúsculas (A, B, C, ...) type="a": Letras minúsculas (a, b, c, ...) type="I": Algarismos romanos maiúsculos (I, II, III, ...) type="i": Algarismos romanos minúsculos (i, ii, iii, ...)

Funcionalidade Tag (Ul)

(Voltar)

A tag (ul) no HTML é usada para criar uma lista não ordenada de itens. Isso significa que os itens da lista não seguem uma ordem específica e são geralmente apresentados com marcadores (também conhecidos como bullets), como pontos, círculos ou quadrados.

Atributo útil da (ul): type (não recomendado nas versões mais modernas do HTML, mas ainda pode ser encontrado em códigos antigos): Define o estilo do marcador. type="disc": Ponto sólido (padrão) type="circle": Círculo oco type="square": Quadrado sólido

Funcionalidade Tag (Li)

(Voltar)

A tag (li) no HTML é usada para definir um item de lista. Ela é utilizada dentro de elementos de lista como (ul), (ol) e (menu) para representar cada item individual. Como funciona: A (li) não é usada sozinha; ela deve estar dentro de uma tag de lista, como: (ul): Lista não ordenada (com marcadores) (ol): Lista ordenada (numerada) (menu): Lista de opções, geralmente usada para menus contextuais ou listas de comandos

Funcionalidade Tag (a)

(Voltar)

A tag (a) no HTML é usada para criar hiperlinks, permitindo que os usuários naveguem para outras páginas web, seções dentro da mesma página, endereços de e-mail, arquivos para download, ou outros recursos. Ela é uma das tags mais importantes da web, pois conecta diferentes partes do conteúdo e sites.

Atributos comuns da : href (Hypertext Reference): Especifica o destino do link. Pode ser uma URL absoluta (como https://www.exemplo.com) ou uma URL relativa (como /pagina-interna). Quando o href está vazio (...), o link não leva a lugar algum e é frequentemente usado para adicionar comportamento de clique por meio de JavaScript. target: Define como o link será aberto. Alguns valores comuns são: "_self": Abre o link na mesma aba ou janela (padrão). "_blank": Abre o link em uma nova aba ou janela. "_parent": Abre o link na janela pai do frame (se houver). "_top": Abre o link na janela completa, quebrando qualquer frame

Funcionalidade Tag (hr)

(Voltar)

A tag (hr) no HTML é usada para inserir uma linha horizontal na página. Ela geralmente é usada para separar seções de conteúdo, criando uma divisão visual entre diferentes partes do texto ou elementos.

Características: A (hr) é um elemento vazio, o que significa que não precisa de uma tag de fechamento. Por padrão, ela aparece como uma linha cinza que se estende horizontalmente pela página.

Funcionalidade Tag (sub)

(Voltar)

A tag (sub) no HTML é usada para aplicar texto em subscrito, ou seja, para exibir o texto ligeiramente abaixo da linha base do texto normal. Ela é frequentemente utilizada para fórmulas químicas, anotações matemáticas, e referências, onde é necessário colocar números, letras ou símbolos menores abaixo do texto principal.

Características: O texto dentro de (sub) é exibido em um tamanho menor e fica abaixo do texto normal. Ele é frequentemente usado em conjunto com a tag (sup), que faz o contrário: coloca o texto em sobrescrito (acima da linha do texto).

Funcionalidade Tag (sup)

(Voltar)

A tag (sup) no HTML é usada para aplicar texto em sobrescrito, ou seja, para exibir o texto ligeiramente acima da linha base do texto normal. Ela é frequentemente utilizada para exponenciação matemática, notações científicas, referências, e abreviações ordinais.

A tag (sup) é especialmente útil para escrever de forma clara e precisa expressões que precisam aparecer como sobrescritos. Ela melhora a legibilidade e ajuda a manter o formato correto para contextos científicos, matemáticos, acadêmicos e técnicos.

Funcionalidade Tag (blockquote)

(Voltar)

A tag (blockquote) no HTML é usada para citar um bloco de texto que foi retirado de outra fonte. Ela é utilizada para destacar citações longas e dar um visual distinto ao texto citado, geralmente aplicando recuo e espaçamento extra para diferenciá-lo do restante do conteúdo.

Funcionalidade Tag (font)

(Voltar)

O marcador "Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML. É importante ressaltar que dentro do contexto do HTML 4.01 o marcador esta em desuso e será obsoletado no HTML 5. Atributos da tag : Cor, que é inserida da seguinte forma: Texto Tipo, que é definido das seguintes formas: Com uma única de fonte: Texto

Funcionalidade Tag (del)

(Voltar)

O elemento HTML (del) (ou Elemento HTML de Texto Excluído) representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

Funcionalidade Tag (abbr)

(Voltar)

O Elemento HTML (abbr) (ou Elemento de Abreviação HTML) representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.
